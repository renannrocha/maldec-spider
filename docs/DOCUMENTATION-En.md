# Introduction
The MalDec Labs website is an online platform specialized in malware analysis. Its main objective is to analyze suspicious files submitted by users and determine whether they contain threats.

<picture>
    <img alt="home page MalDec Labs" src="./assets/bannerHomePage.png">
</picture>

Through the MalDec Engine (an artificial intelligence developed by MalDec Labs), it analyzes these files and provides detailed information about their nature and potential risk. Additionally, the platform offers subscription plan options and a marketplace for buying and selling analyses. In summary, MalDec is an essential tool for cybersecurity, helping to protect systems and networks against digital threats.

## Project Overview
The Website-API is a RESTful application responsible for all back-end services of the MalDec Labs website. Through the Website-API interface, it is possible to make requests following all necessary parameters to perform a set of operations that will be described in this documentation.
